基础（三）数据类型

Author: 万叶 Date: June 3, 2022  12:33:18 Category: C/C++

在C语言中，数据类型分成了基本类型、构造类型和指针类型，其构成了如下的关系：

基本数据

整型

C语言中使用整型来存放整数类型的数字。整型可以分成短整型short、整型int、长整型long和长长整型long long。

取值范围

数据类型 取值范围 字节数 输出格式符
short -32768 ~ 32767 2 %hd
int -2147483648 ~ 2147483647 4 %d
long -2147483648 ~ 2147483647 4 %ld
long long -9223372036854775807 ~ 9223372036854775808 8 %lld

在定义整型的时候，需要根据数据的范围确定应该选择什么类型。若范围选小了，容易造成数据溢出。范围选大了，容易造成资源浪费。

无符号整型

在整型前使用关键字unsigned来表示一个无符号的整型的数字。默认情况下定义的整型都是有符号的，只有指明无符号，才表示无符号的整型。从数值上来说，无符号整型的取值范围要比有符号的整型要大一倍。

数据类型 取值范围 字节数 输出格式符
unsigned short 0 ~ 65535 2 %hu
unsigned int 0 ~ 4294967295 4 %u
unsigned long 0 ~ 4294967295 4 %lu
unsigned long long 0 ~ 1844674407370955161 8 %llu

long类型的全称应该是long int，long long类型的全称应该是long long int。

C语言标准中只规定了：long类型不小于int类型长度，int类型长度不小于short类型长度。因此实际过程中的int和long类型取值范围取决于编译器。在16位编译器上，int占2个字节，long占4个，此时int的取值范围就是[-32768~32767]，而long的取值范围就是[-2147483648,2147483647]。在32位和64位的编译器上，int和long都是占4个字节，此时取值范围都是[-2147483648,2147483647]。

char类型

C语言中char类型只能存储单个字符。char占1个字节。从严格意义上来说，char类型其实本质是整型，因为char类型存储的实际上是ASCII码整数。

char类型使用的是单引号定义，输出字符使用%c。char 变量在内存中存储的是字符对应的 ASCII 码值。如果以 %c 输出，会根据 ASCII 码表转换成对应的字符；如果以%d输出，那么还是整数。

美国信息交换标准码（American Standard Code for Information Interchange，简称ASCII）是一套基于拉丁字母的编码，也是最通用的信息交换标准。ASCII码表的范围是0到127，所以7位比特位就足够表示全部的ASCII码表。

实型（浮点型）

实型也被称为浮点型、浮点数、小数。在C语言中，实型被分成了float单精度、double双精度和long double长双精度。在实际开发过程中，建议多使用double类型。

类型 数的精度 字节数 输出格式
float 6-7位的有效数字 4 %f
double 15-16位有效的数字 8 %lf
long double 占用的内存是double的两倍，但表达数据的精度和double相同 16 %Lf

科学计数法

科学计数法不改变数值的大小，而是改变数值的书写形式。科学计数法采用%e或者%E来表示，若需要格式化输出科学计数法，还可以使用%0m.ne或者%0m.nE的形式，此时：

类型转换

C语言术语强类型的语言，不同的类型之间是不允许数据进行运算的，必须保持类型一致才能进行运算。数据类型的转换有两种：隐式类型转换和显式类型转换4。对于隐式类型转换。其实变化的原因主要是因为计算的数值为了匹配计算精度而进行的，一般情况下都是较低精度的变量类型会转变为较高精度的变量类型。而显式类型转换则是我们主动控制了类型的精度，可以抛去我们不需要的高的精度。同时，由于指定了类型，在转换过程中不会产生歧义。

隐式类型转换

C语言是强类型语言，也就是说不同类型的数据之间是不能进行运算的。必须保持一致的类型才能进行运算。也就是说在这个不同数据类型的计算过程中，C语言自动进行了一次类型转换，使得两个变量的数据类型一致，才能进行相关的计算。这种自动的转换，也称之为隐式类型转换。下面的表格就展示了类型转换的规律，当在计算过程中，数值类型不一致的时候，就会发生自动的类型转换，转换的类型是将表格中处于下方的较低优先级的数据类型，向表格上方的较高优先级的数据类型进行转换。

级别 数据类型
1 long double
2 double
3 float
4 unsigned long long
5 long long
6 unsigned long
7 long
8 unsigned int
9 int
10 char short int

根据这个表格我们就可以看出之前的转换中，int 与 float 类型进行计算，编译器会自动将 int 类型转换为 float 类型进行计算。从而使得运算在相同的数据类型间进行。

显式类型转换

如果说隐式类型转换是编译器自动进行的类型转换，那么显式类型转换，则是我们人为的进行数据类型的转换，这里可以理解为是一种强制的类型的转换，这种转换将不再遵守上面的转换规则，而是按照我们人为的标明的类型进行转换。

就是在我们需要指定类型的变量前加上数据类型，并用圆括号包裹。例如：(int)a， (float)b， (long)c等。

C语言中，程序的运行结构有三种：顺序结构、选择结构和循环结构。

sizeof

sizeof是c语言中的一个操作符，类似于++、--等等。sizeof能够告诉我们编译器为某一特定数据或者某一个类型的数据在内存中分配空间时分配的大小，大小以字节为单位。

使用sizeof需要注意的地方有：

  1. sizeof返回的占用空间大小是为这个变量开辟的大小，而不只是它用到的空间。和现今住房的建筑面积和实用面积的概念差不多。所以对结构体用的时候，大多情况下就得考虑字节对齐的问题了；
  2. sizeof返回的数据结果类型是unsigned int；
  3. 要注意数组名和指针变量的区别。通常情况下，我们总觉得数组名和指针变量差不多，但是在用sizeof的时候差别很大，对数组名用sizeof返回的是整个数组的大小，而对指针变量进行操作的时候返回的则是指针变量本身所占得空间，在32位机的条件下一般都是4。而且当数组名作为函数参数时，在函数内部，形参也就是个指针，所以不再返回数组的大小；
